# curve-js-sdk

A JS SDK companion for curve

## Develop

To develop this JS-SDK on your own, simply run the following commands to install:

```bash
npm i
```

Then to develop, run:

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to NPM

[major] `npm version major`

[minor] `npm version minor`

[patch] `npm version patch`
